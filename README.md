# Project title: Employee Attrition Dashboard
Interactive Power BI dashboard analyzing employee attrition trends using demographic, salary, tenure, and job-related factors for actionable HR insights.

# Description
A dynamic Power BI dashboard using CSV data to analyze employee attrition patterns. The analysis focused on key factors such as age, tenure, monthly income, salary, job role, department, field, gender, and age group. Built interactive visuals and filters to identify trends, uncover reasons behind employee exits, and support data-driven HR decision-making through clear, actionable insights.

# Power BI Project Tools & Components
Power BI Desktop: Primary data visualization tool used to design and build interactive reports and dashboards.
Power Query: Used for data cleaning, transformation, and reshaping to prepare raw data for analysis.
DAX (Data Analysis Expressions): Applied for creating calculated measures, enabling dynamic visuals, and implementing analytical logic.
Data Modeling: Established relationships between tables (such as reports, snow, and data dictionary) to ensure accurate analysis.
File Formats: .pbix used for dashboard development and .png used for sharing dashboard previews.

# Business Problem
The company faced employee attrition but lacked clear insights into the key drivers behind it. HR needed a structured way to analyze factors like salary, tenure, department, job role, and satisfaction to identify trends and improve employee retention decisions.
